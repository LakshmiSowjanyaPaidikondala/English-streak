# 🔥 English Streak Tracker — PWA

A beautiful, installable web app to track your daily English communication streak — just like the RealLife widget!

---

## 📱 Features
- Daily streak counter with flame animation
- Weekly view (M T W T F S S)
- Activity checklist: Speaking, Listening, Reading, Writing, Vocabulary
- Monthly calendar view
- Best streak & total days stats
- Share your streak
- Works **offline** after first load
- Installable on Android & iOS home screen

---

## 🚀 Deploy to GitHub Pages (Free Hosting)

### Step 1 — Create a GitHub account
Go to [github.com](https://github.com) and sign up (free).

### Step 2 — Create a new repository
1. Click the **+** button (top right) → **New repository**
2. Name it: `english-streak`
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload files
1. Click **uploading an existing file** (or drag & drop)
2. Upload ALL files from this folder:
   - `index.html`
   - `style.css`
   - `app.js`
   - `sw.js`
   - `manifest.json`
   - `icons/` folder (with icon-192.png and icon-512.png)
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to your repository → **Settings** tab
2. Scroll to **Pages** (left sidebar)
3. Under **Source** → select **Deploy from a branch**
4. Branch: **main**, folder: **/ (root)**
5. Click **Save**

### Step 5 — Get your link
After ~1 minute, your app will be live at:
```
https://YOUR_USERNAME.github.io/english-streak/
```

## 📲 Add to Home Screen

### Android (Chrome)
1. Open your link in Chrome
2. Tap the 3-dot menu (⋮)
3. Tap **Add to Home screen**
4. Tap **Add**

### iPhone (Safari)
1. Open your link in **Safari** (must be Safari)
2. Tap the **Share** button (box with arrow)
3. Tap **Add to Home Screen**
4. Tap **Add**

## 🛠️ Customization
- Edit activities in `index.html` (the `.act` divs)
- Change colors in `style.css` (the `:root` variables)
- Modify motivation messages in `app.js` (`motivationMsg` function)
